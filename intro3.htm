Set your browser correctly


The Ringnut Jeopardy page auto-refreshes once every 15 seconds. If the game isn't "advancing" as it refreshes, try this (these are for recent versions of Microsoft Internet Explorer, your browser may differ):

Instructions for setting up your browser:
  • Go to your "Tools" menu and select "Internet Options."
  • Select "General" and then under "Temporary Internet Files" click on "Settings."
  • Under "Check for newer versions of stored pages" select "Every visit to the page."
Don't forget to change the settings back after today's game!

If at any time the game screen refreshes and just turns white, or gives you an error message, please manually refresh the page.
